# Sorting Algorithms (Sort)

This is a collection of sorting algorithms implemented in C.

## Algorithms

- Bubble Sort
- Insertion Sort
- Selection Sort
- Merge Sort
- Quick Sort
- Counting Sort
- Radix Sort
- Bucket Sort
- Shell Sort
- Stooge Sort
- Slow Sort
- Brick Sort

Other algorithms are not still implemented:

- Heap Sort
- Comb Sort
- Gnome Sort
- Cocktail Sort
- Pigeonhole Sort
- Cycle Sort
- Bitonic Sort
- Pancake Sort
- Bogo Sort
- Bogosort
- Smooth Sort
- Strand Sort
- Tree Sort
- Cube Sort
- Odd-Even Sort
